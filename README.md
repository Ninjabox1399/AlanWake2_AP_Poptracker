# Alan Wake 2 Archipelago Poptracker
A poptracker pack for my manual archipelago randomiser for Alan Wake 2, found here: https://github.com/Ninjabox1399/AlanWake2_Archipelago

Features:
- Full in-game maps with (mostly) accurate checks placed on them
- Full item tracker including:
  - Weapons
- Archipelago check sending from the map
- Annotated maps with all the codes and puzzle solutions
